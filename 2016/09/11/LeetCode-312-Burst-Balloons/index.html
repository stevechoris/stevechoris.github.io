[LeetCode] 312. Burst Balloons

9月 11, 2016 发布在 LeetCode

题目描述:

312. Burst Balloons
Given n balloons, indexed from 0 to n-1. Each balloon is painted with a number on it represented by array nums. You are asked to burst all the balloons. If the you burst balloon i you will get nums[left] nums[i] nums[right] coins. Here left and right are adjacent indices of i. After the burst, the left and right then becomes adjacent.

Find the maximum coins you can collect by bursting the balloons wisely.

Note:

(1) You may imagine nums[-1] = nums[n] = 1. They are not real therefore you can not burst them.
(2) 0 ≤ n ≤ 500, 0 ≤ nums[i] ≤ 100

Example:

Given [3, 1, 5, 8]

Return 167

nums = [3,1,5,8] –> [3,5,8] –> [3,8] –> [8] –> []
coins = 3 1 5 + 3 5 8 + 1 3 8 + 1 8 1 = 167

题目大意：

给定n个气球，下标从0到-1。每个气球都涂有一个数字，用数组nums表示。要求你炸掉所有的气球。如果你炸掉气球i可以得到nums[left]*nums[i]*nums[right]枚硬币。这里left、right为i左右相邻的气球。 爆炸后，left和right相邻。计算可以获得的最大硬币数量。

解题思路：

一开始看觉得没有头绪，参考克林[leetcode] 312. Burst Balloons的思路

动态规划（Dynamic Programming）
时间复杂度O(n^3)
主要考虑最后一个爆炸的气球, 用dp[l][r]表示炸掉l到r直接的气球可以获得的硬币数，状态转移方程为：
dp[l][r] = max(dp[l][r], dp[l,m]+dp[m][r]+nums[l]*nums[m]*nums[r])

m依次遍历l到r直接所有气球，把它当做最后一个爆炸的气球，将问题转换为炸掉[l, m)和(m, r]之间所有气球可得最大金币这两个子问题，最后再加上炸掉气球m所得金币，来更新dp[l][r]。

进一步优化

由于采用递归分治法的思想，dp[l][r]会有很多重复性计算，因此可以建立一个数组，在每一个区间[l,r]计算完成后，将dp[l][r]保存到二维数组中，避免后续的重复计算。
